SOROTH WEBSITE

Open home_page.html in any browser.

Main pages:
- home_page.html
- collections_page.html
- cart_page.html
- profile_page.html
- admin_panel.html

How the workflow works:
1. Browse products and add items to cart.
2. Open cart_page.html and place an order.
3. Open admin_panel.html and confirm the order.
4. Stock will reduce only after confirmation.
5. Open profile_page.html to view order history and status.

This project uses browser localStorage, so it works without hosting or a backend server.
If you want a fresh start, clear site data/localStorage in your browser.
